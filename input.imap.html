input.imap package¶

Submodules¶

input.imap.connection module¶

input.imap.connection.open_connection(verbose=False)[source]¶

Function to establish to mail server and login to user account using IMAP4 protocol. Returns the connection object used in establishing IMAP connection to mail server. Takes as argument boolen value which is used to determine whether function works in verbose mode or not.

input.imap.header module¶

input.imap.header.get_mail_header(to_get, range_=True)[source]¶

This function fetches the emails from the IMAP server as per the parameters passed. :param to_get: List of UIDs of the mails to get. Default value is 2000. :param range_: If true, fetches all emails from the first element in to_get, till the newest mail. If false, fetches only the emails with the UIDs present in to_get. Default value is true.

input.imap.header.init_uid_map()[source]¶

To ensure that references are correctly recorded in the JSON file such that there are no references to mails that do not exist and to ease the processing of headers, a map with the string in the Message-Id field of the header to the UID of the mail is required. This function reads the header.json file and adds required entries to the map. :return: A map with the string in the Message-Id field of the header to the UID of the mail
