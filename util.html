util.preprocessing_utils module¶

util.preprocessing_utils.is_outlier(points, thresh=3.5)[source]¶

Returns a boolean array with True if points are outliers and False otherwise.

Parameters:
  • points – An numobservations by numdimensions array of observations
  • thresh – The modified z-score to use as a threshold. Observations with a modified z-score (based on the median absolute deviation) greater than this value will be classified as outliers.
Returns:

A numobservations-length boolean array.

Note

Reference: Boris Iglewicz and David Hoaglin (1993), “Volume 16: How to Detect and Handle Outliers”, The ASQC Basic References in Quality Control:Statistical Techniques, Edward F. Mykytka, Ph.D., Editor.

Union Find with path compression¶

A disjoint-set data structure, also called union-find data structure implements two functions: union(A, B) - merge A’s set with B’s set find(A) - finds what set A belongs to Union with path compression approach: Each node visited on the way to a root node may as well be attached directly to the root node. attach the smaller tree to the root of the larger tree Time Complexity : O(a(n)), where a(n) is the inverse of the function n=f(x)=A(x,x) and A is the extremely fast-growing Ackermann function. Author: NRYoung Source: https://github.com/nryoung/algorithms/blob/master/algorithms/data_structures/union_find_with_path_compression.py
